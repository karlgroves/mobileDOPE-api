# Utilities

This directory contains utility functions and helpers.

## Planned Utilities

- `logger.ts` - Bunyan logger configuration
- `errors.ts` - Custom error classes
- `response.ts` - Standard API response formatting
- `validation.ts` - Validation helper functions
- `crypto.ts` - Encryption/decryption utilities
- `date.ts` - Date formatting utilities
- `pagination.ts` - Pagination helpers

All utilities should:
- Be pure functions when possible
- Be unit tested
- Have clear, descriptive names
- Include JSDoc documentation
